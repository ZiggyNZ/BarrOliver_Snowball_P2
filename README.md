# Snowball
Snowball is a Marble Madness inspired co-operative platform/puzzle game.

## Controls:
* Player 1:
  * Movement: W/A/S/D
  * Jump: Spacebar
  * Camera zoom in: E
  * Camera zoom out: Q
  
* Player 2:
  * Movement: I/J/K/L
  * Dash: Return (movement keys give direction)
  * Camera zoom in: U
  * Camera zoom out: O
